    我是黃暄晴，目前剛畢業於佛光大學資訊應用學系碩士班，因在校成績優異， 透過學校的五年一貫計畫(4+1)，僅念一年碩士，即順利在2019年6月畢業。 求學期間我不斷地在接觸Unity3D引擎與使用C#語言開發小型遊戲。 大一升大二的暑假，曾經在蘇澳冷泉賣票打工，因此發現自己能力尚有很多不足，進而去報名補習班， 激勵自我學習更多資訊相關技術。 大二下學期，便開始跟著我的指導教授參與兩項科技部的專題研究計畫， 以及通過科技部的大專學生研究計畫申請，也在同一時期獲得 經濟部產業人才能力鑑定證書 － 初級行動裝置程式設計師(Android) 與 經濟部產業人才能力鑑定證書 － 初級行動遊戲程式設計師(Android) 兩張Android證照。 大三時，不僅繼續參與科技部的專題計畫，也獲得 Silicon Stone Education – Designing 3D Animation Games Using Unity 與 Silicon Stone Education – Designing 3D Animation Games Using 3D Studio Max 兩張國際認證。 到了大三升大四的暑假，透過經濟部的DIGI + Talent 跨域數位人才加速躍升計畫到資策會數位教育研究所實習半年， 同時取得Microsoft Technology Associate - Software Development Fundamentals (C#)證明。
